# Aula 12 - Métodos em Python

## Tema
`__init__`, `self` e métodos em classes Python.

## Objetivo da aula
Entender que objetos podem guardar dados e também executar ações.

Na aula anterior, usamos uma classe vazia e criamos atributos depois que o objeto já existia.

Nesta aula, vamos organizar melhor o código usando:

- `__init__`: método especial que inicia o objeto com dados.
- `self`: referência ao próprio objeto.
- métodos: funções dentro de uma classe.
